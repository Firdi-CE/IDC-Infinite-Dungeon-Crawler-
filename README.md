# IDC (Infinite Dungeon Crawler)

This is a classic roguelike dungeon crawler game built in Python. It uses the `libtcod` library to handle the display, input, and map logic.

## Tech Stack
* **Language:** Python
* **Library:** `libtcod`

## How to Run
1.  Ensure you have Python 3 and the `libtcod` library installed.
2.  Run the game:
    ```bash
    python game/game/game.py
    ```

*(Note: This project includes the `libtcod` library files directly in the repository for portability.)*
